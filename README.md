UI Clone

## Tech Stack

**Client:** React JS

## Run Locally

Clone the project

```bash
  git clone https://github.com/raghav-03/skytrust.git
```

Go to the project directory

```bash
  cd skytrust
```

## FrontEnd

Hosted Link - https://skytrust-pi.vercel.app/


Install dependencies

```bash
  cd frontend/
  npm install
```

Start the server

```bash
  npm run start
```
